This repository documents the setup process of a development environment using Cursor IDE, along with AI-powered extensions and GitHub integration.

- Cursor IDE  
- Claude Code extension (installed, login requires paid plan)

## Steps Completed

1. Installed Cursor IDE from https://cursor.com/  
2. Opened Cursor and explored the interface  
3. Installed Claude Code extension  
4. Attempted to log in to Claude Code (blocked by subscription requirement)  
5. Installed Codex extension and logged in automatically (existing session)  
6. Created a public GitHub repository  
7. Installed Git and configured it via Terminal  
8. Cloned the repository using Cursor  
9. Opened the repository inside Cursor  
10. Created and edited this README.md file  
11. Configured Git user name and email

13. Pushed changes to GitHub

## Issues & Solutions

### 1. Claude Code Login Requirement
- **Issue:** Claude Code requires a Pro or Max subscription to log in  
- **Decision:** Did not proceed with a paid subscription for this exercise  
- **Outcome:** Extension installed successfully, limitation documented  

---

### 2. Git Not Detected in Cursor
- **Issue:** "Git: Clone" command did not appear in Cursor  
- **Cause:** Git was not installed or not detected  
- **Solution:** Installed Git via Terminal and restarted Cursor  
- **Result:** Git commands became available and repository was cloned successfully  

---

### 3. Git Configuration Required
- **Issue:** Commit was blocked due to missing Git user.name and user.email  
- **Solution:** Configured Git identity via Terminal  
- **Result:** Successfully enabled commit functionality  

---

### 4. Codex Login Behavior
- **Observation:** Codex extension logged in automatically  
- **Reason:** Existing session on the device  
- **Outcome:** Extension was ready to use without additional setup
